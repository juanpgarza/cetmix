Odoo Message Filter & Spam Protection

Filter messages

Block spam

This is an extension for Mail Messages Easy module. Please check its description before using this module!

MAIN FEATURES

  • NB! Only new incoming messages will be filtered. Incoming replies or alias matched will be routed as usual.
    You need to configure "Cetmix Message Filter" as a fallback model in the incoming mail server settings
  • Mark message as spam
  • Post message to model: will create new record (e.g. Lead or Task)
  • Post message to record: will post to existing record
  • Keep here: will be posted to the current filter record
  • Do not receive: message will be not received
  • N.B. This is an extension of Mail Messages Easy module. Please check its description before using this module!

PRO FEATURES

Click to get Pro Version and support the project!

  • Search for record and post: message with subject "Question INV/2021/0750 payments" will be posted to Invoice INV/2021/0750
  • Post to partner: message from "John Smith js@example.com" will be posted to "John Smith" partner record in Odoo
  • Modify message: replace recipients (e.g "Bob" with "Mike"), add prefix (e.g. "TODO:") or append any text to subject or body
